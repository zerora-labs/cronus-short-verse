# CronusShortVerse 规则

## 1. 天体分类

每个叙事实体（角色、事件、地点）根据其**叙事质量与生命周期**被归为五种类型之一。

| 类型     | 质量           | 生命周期       | 示例                         |
| -------- | -------------- | -------------- | ---------------------------- |
| **恒星** | 非常高         | 整个系列       | 主角                         |
| **行星** | 高             | 大部分系列     | 常驻配角                     |
| **彗星** | 中等           | 1-3 集         | 客串角色、临时剧情工具       |
| **流星** | 低             | 单集或单个场景 | 一次性谣言、闪回             |
| **黑洞** | 可变（但奇异） | 永久谜团       | 秘密、未解决的创伤、禁忌知识 |

**规则 1.1**：一颗恒星可以有多颗行星绕其公转。行星可以有卫星（次要角色），但卫星不在核心模型中追踪。

**规则 1.2**：如果一颗彗星出现在超过 3 集中，它可以变为行星，其质量相应增加。

**规则 1.3**：流星不留下永久轨道。它可能暂时影响行星或恒星的轨迹。

**规则 1.4**：黑洞永远不会被完全揭示。其影响能被感受到（角色被吸入、事件螺旋下降），但其本质始终未知。

## 2. 引力相互作用

- **吸引**：具有强烈叙事联系的两个角色（如英雄与反派）相互施加引力。
- **排斥**：目标对立的角色相互推开。
- **轨道**：始终跟随另一个角色的角色处于绕行状态。
- **碰撞**：当两颗恒星直接冲突时，结果是一个重要的剧情节点。

**规则 2.1**：引力的大小与两者质量的乘积成正比，与叙事距离的平方成反比。

**规则 2.2**：黑洞可以吸收任何靠得太近的天体。被吸收的实体从活跃宇宙中移除（例如角色离开或死亡）。

## 3. 时间与循环

- **集**：最小时间单位。每集约 2 分钟。
- **篇章**：一组集（通常 10-20 集）构成一个完整的故事循环。
- **季**：多个篇章共享一个总体主题。
- **循环重置**：在一个篇章结束时，一些轨道可能会重置（角色回到默认位置），而另一些则继续。

**规则 3.1**：每集结尾的悬念会增加下一集的叙事速度。

**规则 3.2**：允许时间循环。当循环发生时，相同角色重复动作，但每个循环有微小的轨道偏移。

## 4. 噪声与信号

并非短剧中的每个事件都具有叙事意义。**噪声过滤器**区分：

- **信号**：改变角色状态、关系或剧情方向的事件。
- **噪声**：填充内容、背景活动或不影响主叙事的事件。

**规则 4.1**：只有信号事件被记录在宇宙历史中。噪声事件被丢弃。

**规则 4.2**：流星通常是噪声，除非它触发了信号事件。

## 5. 可扩展性

创作者可以定义新的天体类型（例如：星云表示神秘区域，脉冲星表示反复出现的惊喜元素），只要它们尊重引力模型。
